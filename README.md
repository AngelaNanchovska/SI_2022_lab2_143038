<h2>Цикломатска комплексност</h2>

Цикломатската комплексност може да ја пресметаме со формулата: E-N+2

N - број на темиња = 20 </br>
E - број на ребра = 27 </br>
27-20+2 = 9 </br>
Цикломатската комплексност ќе биде 9.
